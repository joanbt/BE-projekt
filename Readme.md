### Instalacja

```
 $ docker-compose build
 $ docker-compose run php composer install

```

### Run

```
 $ docker-compose up
```

### adres sklepu
	
	http://127.0.0.1:8880/

### Magento Admin

	http://127.0.0.1:8880/admin/

### phpMyAdmin

	http://127.0.0.1:8080/

> Wszystkie dane do logowania znajdują się w pliku .env
